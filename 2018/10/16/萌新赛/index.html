##web##

1.根本想不到

http://39.106.148.174:9088/

解题思路：这道题是要登录wordpress网站的后台，WordPress网站的登陆地址一般都是:http://你的域名/wp-admin.php 或者 http://你的域名/wp-login.php，然后就试了一下这两个，发现输入/wp-admin.php就出现了flag。

###2.萌萌的小脸真可爱啊###
http://39.106.148.174:9096/
这道题是任意文件下载
解题思路：首先查看页面源代码

<!DOCTYPE html>
<html lang="en">
    <head>
        <meta charset="utf-8">
        <title>Yuanyuande! Hin kenai~~</title>
        <style type="text/css" media="screen">

        </style>
    </head>
    <body>
        <center>
            <embed src="xiaogangji.mov" width="400px" height="500px" quality="high" />
            <p class="pic">圆圆的小刚几敲级可耐的嘞~~~</p>
            <p class="pic" align="center">
                <a href="download.php?url=Y3V0ZS5qcGc=" target="_blank" align="center">萌萌哒</a>
            </p>
        </center>
    </body>
</html>

点开这个<a href="download.php?url=Y3V0ZS5qcGc=" target="_blank" align="center">萌萌哒</a>我们看到了一张图片，看到它的url=Y3V0ZS5qcGc=

我们将它的url进行bs64解码，得到cute.jpg

尝试将url改为flag.jpg经过bs64加密后的ZmxhZy5qcGc=

发现得到的这个flag是不正确的，然后在仔细分析一下代码，我们尝试将download.php进行bs64加密得到ZG93bmxvYWQucGhw

然后回车后自动下载一个文件用notepad++打开后得到
<?php

error_reporting(0);

$url =base64_decode($_GET[url]);

if ($url=='cute.jpg'|| $url == 'download.php' || $url=='fllllagg.php'|| $url == 'flag.jpg') {
    $file_size = filesize($url);
    header ( "Pragma: public" );
    header ( "Cache-Control: must-revalidate, post-check=0, pre-check=0" );
    header ( "Cache-Control: private", false );
    header ( "Content-Transfer-Encoding: binary" );
    header ( "Content-Type: image/jpeg");
    header ( "Content-Length: " . $file_size);
    header ( "Content-Disposition: attachment; filename=".$url);
    echo(file_get_contents($url));
    exit;
}
else{
    echo "403 Forbidden";
}

我们发现了这个$url==’fllllagg.php’，我们将fllllagg.php进行bs64加密后得到ZmxsbGxhZ2cucGhw

然后又自动下载一个文件，以上面同样的方式打开后得到flag。
