爬取校网新闻

发表于 2018-08-04 | 更新于 2018-08-08

网络爬虫—–现在只简单的了解了一下如何爬取页面上的信息，具体的还有待学习，我就简单的写一下爬取学校官网一部分新闻的吧。。。也并不是自己全写的，参照了网上的脚本。。。

1.先补充一点pycharm库文件的知识

  1. 使用urllib:在Python2版本中，有urllib和urlib2两个库可以用来实现request的发送。而在Python3中，已经不存在urllib2这个库了，统一为urllib。urllib包括了四个模块，如下：
  • urllib.request可以用来发送request和获取request的结果
  • urllib.error包含了urllib.request产生的异常
  • urllib.parse用来解析和处理URL
  • urllib.robotparse用来解析页面的robots.txt文件

使用urllib.request发送请求
urllib.request.urlopen()基本使用
urllib.request 模块提供了最基本的构造 HTTP 请求的方法，利用它可以模拟浏览器的一个请求发起过程，同时它还带有处理 authenticaton （授权验证）， redirections （重定向)， cookies (浏览器Cookies）以及其它内容。我们来实际操作一下，首先我们就来将校网的页面抓下来。
import urllib.request
response = urllib.request.urlopen('http://www.zzti.edu.cn/')
print(response.read().decode("UTF-8"))
通过这几行代码，我们就可以抓取到校网主页的源代码

##2.抓取校网新闻代码##

我要抓取的就是官网页面上的校内通知的这块，我需要把它的页面新闻题目和内容抓取下来，代码如下：

import urllib.request
import requests
from bs4 import BeautifulSoup
import re
import bs4#导入必要的库文件
raw = 'http://www.zzti.edu.cn/index/tzgg'#打开文件用于写入新闻页面url（url 即请求链接）
with open("20.txt","a+") as f:
    urls = []
    for i in range(220,0,-1):
        if i == 220:
            url = raw + ".htm"
        else:
            url =raw + "/" + str(i) + ".htm"
        res = urllib.request.urlopen(url)
        r = res.read().decode("UTF-8")

        soup = BeautifulSoup(r,"html.parser")
        a1 = soup.find_all('a',class_='c67214')
        for a in a1:
            h = a.get('href')
            # 获得链接并对改版前后做不同处理
            if h[3] == '.':
                temp = 'http://www.zzti.edu.cn/' + h[6:] + '\n'
            else:
                temp = 'http://www.zzti.edu.cn/' + h[3:] + '\n'
            if temp not in urls:
                urls.append(temp)
                f.write(temp)
    f.close()
count = 0
with open("20.txt","r") as d:
    for line in d.readlines():
        line = line.strip()
        count += 1
        response = urllib.request.urlopen(line)
        a =response.read().decode("UTF-8")
        soup = BeautifulSoup(a,"html.parser")
        s = soup.find_all('div',id=re.compile("vsb"))
        with open("内容.txt","a+",encoding='utf-8') as c:
            for i in s:
                c.write(i.get_text())
        c.close()
        print("第%d篇爬取成功"%(count))
    d.close()

print('DONE')

运行后出现20.txt和内容。txt两个文件夹，分别是爬取到的新闻链接和新闻内容。
